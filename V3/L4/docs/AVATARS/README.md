# L4 Avatar System

> **202 Sacred Characters** — 51 core playable/NPC avatars + 151 extended roster across 17 cultural circles.

---

## Overview

Every avatar in OASIS is a **sacred archetype** with:
- Unique quest line (5 quests each = 255 core quests total)
- Teaching / philosophy
- Special ability
- Consciousness Level (CL) requirement
- NFT rarity tier
- Ray / tradition affiliation

## In-Game Mechanics

- **Quest unlock:** Each avatar has 5 quests. Complete all 5 = "Avatar Mastery" bonus (+10,000 XP).
- **Ability slot:** Players equip 1 avatar ability at a time (cooldown-based).
- **Ray synergy:** Guilds with members of all 7 rays get +15 % territory bonus.
- **Teaching echoes:** Avatar dialogues are voice-acted snippets tied to real scripture / philosophy.
