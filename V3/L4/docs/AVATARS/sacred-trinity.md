## Key Mechanics

- **Ramayana Key (1/3):** Collected by completing Rama + Sita + Hanuman + Radha quest lines.
- **Mahabharata Key (2/3):** Collected by completing Arjuna + Bhima + Mama Yashoda + Vishwakarma + Vyasa-Kamil quest lines.
- **Unity Key (3/3):** Requires both previous keys + CL 7. Unlocks Krishna-Maitreya final encounter.
- **Ray Synergy:** Hindu avatars span Blue (Will), Pink (Love), Yellow (Wisdom), Ruby-Gold (Service), and White (Purity) rays.
